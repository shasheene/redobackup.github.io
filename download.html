It is a live environment, so you cannot run the program from within Windows or any other operating system; you need to follow the steps below exactly as they are presented in order to use Redo Backup.

Getting Started

1. Download the latest live image

Download the latest version of the Redo Backup live image from here.

2. Write the image to a USB

You'll need to write the image to a USB using a program such as balenaEtcher which works on Windows, Mac and Linux.

Using Unetbootin is not recommended, because it prevents the Redo Backup splash screen, help files, boot menu messages, and other features from being displayed or used.

3. Reboot your computer

Redo Backup does not run from inside Windows or any other operating system because it is one! You must restart your computer with the USB inserted in order to use Redo Backup. To boot from the USB, you may need to press a key such as F8 or F12 while your computer is starting up.

After the USB boots, you will have a mini operating system loaded into memory which will launch Redo Backup. It will not install anything to your hard drive. You decide when and where you want to save any backup images, if you choose to do so.

System Requirements

  • PC (Intel/AMD compatible)
  • 1GB RAM

Installing to a CD or DVD (Optional)

Redo can boot from CD or DVD, but it's not recommended. USB drives are much faster.

1. Download the latest live image

Download the latest version of the Redo Backup live image from here.

2. Reconsider your need to boot from CD or DVD

Booting from CD or DVD is slow. Prefer USB drives if possible.

3. Burn the image to a DVD

Burn the image using a program like InfraRecorder.
